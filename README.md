## 기여 방법

1. Node.js와 npm 준비
  - [nvm을 사용한 설치 방법](https://github.com/nvm-sh/nvm#installing-and-updating)

2. 로컬 개발환경 세팅

```sh
# 1단계: 저장소 클론
git clone https://github.com/lipcoding-kr/lipcoding-kr.github.io.git

# 2단계: 클론받은 저장소 위치로 이동
cd <YOUR_PROJECT_NAME>

# 3단계: 필요한 패키지 설치
npm i

# 4단계: 로컬 개발 서버 실행
npm run dev
```
